## 安裝指南

1. Clone `config.toml.example` as `config.toml`：
    ```bash
    cp config.toml.example config.toml
    ```

2. Clone `.env.example` as `.env` and fill your sensitive info. ：
    ```bash
    cp .env.example .env
    ```

3. Install dependencies：
    ```bash
    pip install -r requirements.txt
    ```
